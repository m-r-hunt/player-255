Notes

N+ is an expanded version of a classic flash game: N.


The core of the game is the same: tight single screen platforming levels, on a constant ticking timer. Good controls, a small but deep set of game objects combined in many ways to overcome.


The only criticisms I have are that certain levels are just "do the same kinda hard thing 10 times in a row" which gets annoying. Save states really help even out the difficulty. I also hated the "springy box" element, never felt right to me.


Maybe overstays its welcome a little bit, and I did lean more and more on save states late in the game. I cheated slightly on my completion, beat all the original levels, more are unlocked afterwards but I ignored them and got the credits from the main menu.
