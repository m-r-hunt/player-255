Screenshots

Notes

This is a strategy RPG. It has one really unique feature which is that you attack by moving through enemies, potentially multiple in one turn.


The game is a bit clunky and production values are a bit low. The plot is the most generic SRPG plot about kingdoms and princes.


If history has demonstrated anything it's that I don't have the patience to play SRPGs when they aren't the best of the best.
